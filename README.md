# Reminder Flow - Sistema de Recordatorios de Facturas

Sistema completo para gestión de recordatorios de facturas con Backend, Frontend y Mobile.

## Diagrama

Se incluye un diagrama de la implementación de la aplicación en el archivo `Diagrama_Aplicacion.png`.

## Ejecución

### Backend
```bash
cd Test_Reminder_Flow_Monolegal_Backend
docker-compose up --build
```
API disponible en: http://localhost:8000

### Frontend
```bash
cd Test_Reminder_Flow_Monolegal_Frontend
npm install
npm start
```
App disponible en: http://localhost:4200

### Mobile
```bash
cd Test_Reminder_Flow_Monolegal_Mobile
flutter pub get
flutter run
```
